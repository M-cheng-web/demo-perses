# @grafana-fast/types

TypeScript 类型定义，包含 Dashboard、Panel、Query 等核心类型。

## 安装

```bash
npm install @grafana-fast/types
```

## 使用

```typescript
import type {
  Dashboard,
  Panel,
  PanelGroup,
  Query,
  TimeRange
} from '@grafana-fast/types'
```

## 主要类型

- **Dashboard**: 仪表板配置类型
- **Panel**: 面板配置类型
- **PanelGroup**: 面板组配置类型
- **Query**: 查询配置类型
- **TimeRange**: 时间范围类型
- **DataSource**: 数据源类型
- **Chart**: 图表配置类型
